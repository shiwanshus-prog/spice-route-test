# 🌶️ Spice Route — Indian Food Website

A single-page, all non-vegetarian Indian restaurant website built with plain HTML, CSS,
and JavaScript. No frameworks, no build step — open `index.html` in a browser and it runs.

## Features
- Sticky navbar, hero banner, about, and contact sections
- Filterable menu of **23 non-vegetarian dishes** across 6 categories
- Real food photos with an emoji fallback
- Fully responsive (mobile, tablet, desktop)
- Published visiting hours: **Mon–Sun, 9:00 AM – 9:00 PM**

## Run locally
Double-click `index.html`, or serve the folder with any static server.

## Files
| File          | Purpose                                   |
|---------------|-------------------------------------------|

| `styles.css`  | Styling, layout, animations, responsive   |
